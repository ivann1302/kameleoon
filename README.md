# Interactive Line Chart

Интерактивный график конверсий с фильтрацией вариаций и настройками отображения.

### Инструкции к запуску проекта

```bash
npm install
```

### Режим разработки

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173`

### Сборка

```bash
npm run build
```
Собранные файлы будут находиться в папке `dist/`


### Основной функционал
- **Conversion Rate Chart** - отображение графика конверсий для всех вариаций, все значения в процентах
- **Hover Interaction** - при наведении показывается вертикальная линия и всплывающее окно с дневными данными
- **Minimum Selection** - минимум одна вариация всегда должна быть выбрана
- **Auto-scaling Axes** - при переключении вариаций оси X и Y автоматически адаптируются к видимому диапазону данных
- **Percentage Display** - все значения отображаются в процентах
- **Responsive Layout** - адаптивная верстка для экранов от 671px до 1300px
- **Variations Selector** - выбор линий для отображения (чекбоксы с цветовыми индикаторами)
- **Day / Week Selector** - переключение между дневным и недельным видом данных

## Дополнительные опции

- **Zoom / Reset Zoom** - интерактивное масштабирование через Brush компонент с кнопкой сброса
- **Line Style Selector** - выбор стиля графика: Line (линейный), Smooth (сглаженный), Area (заполненный)
- **Light / Dark Theme Toggle** - переключение между светлой и темной цветовой схемой
- **Export Chart to PNG** - экспорт графика в PNG изображение

## Стек

- React 19
- TypeScript
- Vite 
- Recharts (для построения графика)
- html2canvas (для  экспорта графика в формате PNG )
